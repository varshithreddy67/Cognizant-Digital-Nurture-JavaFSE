Angular Hands-On 1 - Project Structure Notes

1. angular.json
Contains the Angular workspace configuration such as build, serve, test, assets, styles, and project settings.

2. tsconfig.json
Main TypeScript configuration file shared across the Angular project.

3. tsconfig.app.json
Contains TypeScript compiler settings specifically for the application source code.

4. package.json
Stores project metadata, dependencies, scripts, and package versions.

5. src/main.ts
The entry point of the Angular application that bootstraps the root application.

6. src/app/app.config.ts
Contains the standalone Angular application configuration, providers, and routing setup.

7. src/app/app.ts
Defines the root standalone component of the Angular application.

8. src/index.html
The main HTML page that loads the Angular application inside the browser.
